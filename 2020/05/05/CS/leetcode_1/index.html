【leetcode】验证二叉搜索树

leetcode刷题

题目

给定一个二叉树，判断其是否是一个有效的二叉搜索树（Binary Search Tree, BST）。

假设一个二叉搜索树具有如下特征：

  • 节点的左子树只包含小于当前节点的数。
  • 节点的右子树只包含大于当前节点的数。
  • 所有左子树和右子树自身必须也是二叉搜索树。

示例：

输入: [2 1 3]

输出: true

输入：[5,1,4,null,null,3,6]

输出: false

代码

BST的中序遍历序列为一个递增的有序序列，由此可判断。
